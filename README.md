# Noor Rai - Work Rota

A static, shareable work rota page for May-August 2026.

The page is generated from `noor_work_rota_may_aug_2026.md` and is intended as a human-readable rota view. Please verify the rota against the original calendar before using it clinically or officially.

## Local preview

Open `index.html` in a browser, or serve the folder with any static file server.

## Custom days

The page includes an in-browser custom day planner for vacation days and teaching time. These edits are saved in the visitor's browser with `localStorage`; they do not change the source rota for other visitors.

## Publish

This repo is ready for GitHub Pages. Set the Pages source to the `main` branch and `/root` directory.
